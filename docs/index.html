🌾 FarmHud

Transparent HUD overlay for efficiently farming herbs, ore, and gathering nodes

⬇ Download 📂 View Source

✨ Features

🗺️ HUD Overlay

Transparent minimap overlay centered on screen. See gathering nodes without opening the map.

📍 TrailPath Module

Drop pins to mark your farming route. Pins remain static at world positions with configurable fade timer.

🔌 Addon Integration

Works with GatherMate2, Routes, HandyNotes, LootCollector, and _NPCScan.Overlay.

⚙️ Customization

Configurable keybindings, adjustable HUD size, cardinal direction indicators, mouse interaction toggle.
